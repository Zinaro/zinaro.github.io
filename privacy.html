Project Automation Bot

This application is a personal automation tool designed to bridge Telegram messaging with YouTube channel management. It facilitates the automated upload of video content provided by the authorized developer via a private Telegram bot to their personal YouTube channel.

Note: This is a private project and is not intended for public use.

Privacy Policy

This application respects your privacy and is committed to protecting any data it processes. Since this is a personal-use application:

  • Data Collection: No personal identification information is collected from third-party users.
  • API Access: The application only accesses the Google/YouTube account explicitly authorized by the developer.
  • Data Sharing: No data is ever shared with third parties. All interactions occur directly between the authorized Telegram bot and Google's official API servers.
